# Image guide — Total Klean

Drop your own photos in these folders using the exact filenames below.
The site code already references these names, so once you replace a
file (same name, same folder), it will appear automatically — no code
changes needed. Recommended formats: .jpg or .webp, landscape unless noted.

## /images/logo
- `logo-icon.svg` — the round logo mark (icon only, square/transparent)
- `logo-full.svg` (optional) — full horizontal logo with wordmark

## /images/hero
- `home-hero.jpg` — full-screen hero background on the homepage (1920x1080+, landscape, car in motion or hero shot)
- `booking-hero.jpg` — background for the booking page header
- `services-hero.jpg`, `about-hero.jpg`, `pricing-hero.jpg`, `blog-hero.jpg`, `contact-hero.jpg`, `testimonials-hero.jpg` — optional page header backgrounds

## /images/services
- `service-exterior.jpg` — exterior wash & polish
- `service-interior.jpg` — interior detailing
- `service-ceramic.jpg` — ceramic coating / protection
- `service-restoration.jpg` — paint correction / restoration
- `service-mobile.jpg` — mobile / on-location service

## /images/gallery
- `gallery-01.jpg` through `gallery-08.jpg` — before/after & work shots
  (used on the homepage gallery preview and a future full gallery page)

## /images/team
- `team-01.jpg`, `team-02.jpg`, `team-03.jpg` — team member portraits (square, 1:1)

## /images/blog
- `blog-01.jpg`, `blog-02.jpg`, `blog-03.jpg` — cover images for blog posts
  (1200x800 landscape recommended)

## /images/testimonials
- `client-01.jpg`, `client-02.jpg`, `client-03.jpg` — client headshots (square, 1:1)

---
Until you add your own images, the site uses Unsplash placeholders so
every page already looks complete. Just drop matching filenames in to
replace them.
